EarthumYBot Commands List

=> Hello and welcome to the official EarthumYBot commands list.

=> If you're stuck on a command, you can check the commands below.

=> The current prefix is a..





Fun Commands

Description: These are commands for fun and entertainment.


Note: [] = needed, () = optional, || = or

Commands Description of the Commands Usage
8ball Bot plays a classic game of widsom a.8ball [question]
ascii Bot sends an ASCII version of what what word you sent a.ascii [text]
bingo Bot plays a classic game of bingo a.bingo
coinflip Bot flips a coin a.coinflip
dice Bot rolls a dice a.dice
hello Bot messages "Hello there!" a.hello
killme Bot kills the user then respwans a.killme
math Bot will do the maths a.math [problem]
meme Bot sends a meme a.meme
say Bot says something you sent a.say [text]
tableflip Bot flips a table a.tableflip
unflip Bot unflips a table a.unflip

Management Commands

Description: These are the commands for management.


Note: [] = needed, () = optional, || = or

Commands Description of the Commands Usage
